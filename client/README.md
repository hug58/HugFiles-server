# HugFiles-Client
file server with client synchronization using Flask, Socketio, Celery and Redis. Like dropbox

## INSTALL DEPENDENCIES
```bash
pip install -r requirements.txt
```

## RUN CLIENT 
```bash
python app.py 
```
